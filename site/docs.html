> DOCS // COME FUNZIONA

OSSERVATORIO_SEO è un hub giornaliero di news SEO e AI aggiornato automaticamente ogni mattina alle 07:00 Europe/Rome. Una pipeline Python legge un insieme di fonti autorevoli, filtra le notizie fresche, le riassume in italiano tramite un modello AI (Gemini 2.0 Flash) e pubblica il risultato come pagina statica e come file feed.json pubblico.

Il flusso di ogni run è:

  1. [1] FETCH · scarica in parallelo ~21 fonti (RSS, scraper HTML, Playwright per siti anti-bot)
  2. [2] DOC_WATCH · controlla 9 pagine di documentazione ufficiale e genera un diff se qualcosa cambia
  3. [3] NORMALIZE · pulisce URL, dedup per URL e per fuzzy title, filtra news > 72h
  4. [4] SUMMARIZE · per ogni item l'AI genera titolo italiano, riassunto, categoria, tag, importance 1-5
  5. [5] RANK · score per top10 e raggruppamento in 8 categorie
  6. [6] PUBLISH · scrive feed.json, archivio giornaliero, push su GitHub, deploy Cloudflare Pages

Costo AI tipico per run: €0.005. Tutto gira gratis su GitHub Actions + Cloudflare Pages. Zero database.

> RANKING & CATEGORIE

Ogni item riceve uno score calcolato così:

score = importance * 10       # decisione AI, 1..5
      + source.authority      # config/sources.yml, 1..10
      + freshness_bonus       # 5 se <6h, 2 se <24h, 0 altrimenti
      + doc_change_bonus      # 20 se è un cambio Doc Watcher
      + category_bonus        # 10 google_docs_change, 5 google_updates, 3 ai_models

Le 8 categorie sono: google_updates, google_docs_change, ai_models, ai_overviews_llm_seo, technical_seo, content_eeat, tools_platforms, industry_news. Le etichette Italiane vengono mostrate nel frontend.

  • HOSTING · Cloudflare Pages (statico, noindex)
  • FRONTEND · HTML + Tailwind CDN + vanilla JS, zero build step
  • STORAGE · zero database: tutto committato nel repo git (feed + archivio + stato doc watcher)
