Nuova posizione per i file degli intervalli IP dei crawler di Google

Fonte: Google Search Central Blog

Google ha annunciato lo spostamento dei file JSON contenenti gli intervalli IP dei suoi crawler. La nuova posizione è developers.google.com/crawling/ipranges/, più generica rispetto alla precedente. I file rimarranno disponibili nella vecchia directory (/search/apis/ipranges/) per un periodo transitorio di sei mesi, durante il quale è consigliabile aggiornare i sistemi.

Google ha spostato la posizione dei file JSON che contengono gli intervalli IP dei suoi crawler. Gli intervalli IP sono utilizzati per verificare che le richieste provengano effettivamente dai bot di Google e non da user agent falsificati. La nuova posizione suggerisce che Google sta consolidando la documentazione sui crawler sotto un ombrello più ampio, non limitato alla Search. Questo potrebbe indicare una futura unificazione della gestione degli IP per crawler di diversi servizi Google (Search, News, AdsBot, ecc.), anche se al momento non ci sono conferme ufficiali in tal senso.

Il metodo corretto resta la reverse DNS lookup (verificare che l'hostname termini con .google.com o .googlebot.com), come documentato da Google. Gli intervalli IP sono utili per ottimizzazioni, non per sicurezza primaria.
